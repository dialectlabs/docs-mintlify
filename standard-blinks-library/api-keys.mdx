There are two ways to request your API keys for the Standard Blinks Library:

### Option 1: Through the Dialect Dashboard

If you're already registered, sign in to the [Dialect Dashboard](https://dashboard.dialect.to/), navigate to [API Keys](https://dashboard.dialect.to/api-keys), and click **Get API Keys** on the Blinks card.

### Option 2: Fill out the request form directly

In the future, requesting API keys will be only available through our Dashboard. For now, we still offer you to fill out the [API key request form](https://form.typeform.com/to/uIBkRDsF?utm_source=docs&utm_medium=api-keys-sbl) directly — no dashboard account required.

## Need Help?

If you have questions about API keys or need assistance, reach out on [Discord](https://discord.gg/saydialect).
