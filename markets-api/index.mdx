The Dialect Markets API provides real-time lending market data for Solana's top lending protocols: **MarginFi Lending** and **Kamino Lending**. This powerful endpoint delivers all the market information you need to build lending applications without integrating multiple protocol SDKs.

## What it does

The Markets API delivers a JSON document (6,000+ lines) containing:
- **Real-time lending rates** - Current deposit and borrow APYs for all markets
- **Market liquidity data** - Total deposits, borrows, and available capacity
- **Token information** - Token addresses, symbols, and decimal precision
- **Market limits** - Maximum deposit and borrow amounts per market
- **Ready-to-use Blinks** - Pre-built transaction URLs for deposits and withdrawals
