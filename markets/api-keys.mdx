<Note>
  Code examples in this documentation use a `pk_demo` client key. This key is
  heavily rate limited and intended only for the API Playground. Request your
  own key for production use.
</Note>

There are two ways to request your client keys for the Markets and Positions APIs:

### Option 1: Through the Dialect Dashboard

If you're already registered, sign in to the [Dialect Dashboard](https://dashboard.dialect.to/), navigate to [API Keys](https://dashboard.dialect.to/api-keys), and click **Get API Keys** on the Markets & Positions card.

### Option 2: Fill out the request form directly

In the future, requesting client keys will be only available through our Dashboard. For now, we still offer you to fill out the [client key request form](https://form.typeform.com/to/uIBkRDsF?utm_source=docs&utm_medium=api-keys-sbl) directly — no dashboard account required.
